##  Usage Guide

### Adding a Subject
1. Navigate to **Subjects** section
2. Click **Add Subject** button
3. Fill in subject details
4. Click **Add Subject** to save

### Creating a Schedule
1. Go to **Schedule** section
2. Click **Add Class** button
3. Select subject, day, time, location, and type
4. Click **Add Class** to save

### Managing Tasks
1. Visit **Tasks** section
2. Click **Add Task** button
3. Enter task details
4. Use filters to view specific categories
5. Check the checkbox to mark tasks complete

### Viewing Analytics
1. Navigate to **Analytics** section
2. View summary statistics
3. Explore charts for insights

### Customizing Settings
1. Go to **Settings** section
2. Toggle theme between Light and Dark
3. Adjust preferences
4. Export data for backup
